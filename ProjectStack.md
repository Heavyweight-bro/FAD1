# Project Stack

## Hosting / Deploy

- **GitHub**: source control (цей репозиторій)
- **Vercel**: deploy `apps/web` (React + Vite)
- **Supabase**: Postgres + Auth + Storage + Edge Functions

## Frontend

- React 18
- Vite
- Tailwind CSS
- React Router
- TanStack Query

## Backend

- Supabase Edge Functions (Deno + TypeScript)
- Supabase Postgres (SQL migrations у `supabase/migrations`)
